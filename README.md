# Nemesis Engine

Nemesis is a Vulkan rendering engine written in Rust that simultaneously renders arbitrary
3D meshes and instanced molecular structures from PDB data; built for scientific visualization of
MOGAD-relevant proteins.

## Dependencies

- [Rust](https://rustup.rs) (stable)
- [Vulkan SDK](https://vulkan.lunarg.com/) 1.3+
- [glslc](https://github.com/google/shaderc)

## Building

```bash
git clone https://github.com/nicholasgallina/nemesis.git
cd nemesis
cargo run
```
